musicNet

musicNet is a system for conducting complex searches of symbolic polyphonic music databases. Searches are constructed by buildings networks of relationships and can be multidimensional, searching for patterns in multiple voices and other musical parameters. The database can import and export from music21. In the current version searches are constructed using Python scripts.

A recent version of the musicNet documentation is also available.
